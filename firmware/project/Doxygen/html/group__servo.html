Detailed Description

API for initializing and driving the PWM servo.

This module contains functions to setup Timer2/OC1 for 50 Hz PWM, move the servo to 0 ,90 ,180 degrees, and arbitrary angles,
plus a toggle helper.

void servo_init ( )

Initialize Timer2 and OC1 for PWM control of the servo.

Sets up 20 ms period (50 Hz) channel on RB5 / OC1. Called before any move or toggle.
